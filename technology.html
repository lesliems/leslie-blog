HTML stands for Hyper Text Markup Language.
Hyper Text means text with links in it.
Markup Language means a language used to make text do more than sit on a page.
For example it can turn into an image,links, tables,lists,and much more.
HTML is mainly used to create web pages.

CSS is short for Cascading Style Sheets. CSS can be described as the
skin and make up that cover the bones of HTML.
In other words CSS makes web pages look pretty.

BOOTSTRAP is like a free collection of tools and cool features to add on
your web page. BootStrap can make your web page look more professional.
With bootstrap the sky is the limit

Javascript is a programming language that is used to "create interactive effects within a web browser".
For example, snake is an interactive game on a web page. You can click on the link to see a
picture of itClick here

JQuery is a JavaScript library. It is similar to css but it "saves you a lot of typing"

Melon JS is an HTML5 javascript game engine.In other words, melon js allows javascript games to be played and interacted on a web browser.
